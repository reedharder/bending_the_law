Bending the Law: an open source web platform (akin to, but different from Lexis-Nexis or WestLaw) that allows users to navigate the corpus of SC cases (1950-2001) using the citation and topical (i.e., topic model based) multi-network. Users enter the geometric space through a pre-identified case, which itself may be generated through keyword searches on publicly available or proprietary databases. The user interface will introduce users to cases in the vicinity of a pre-identified case (after specifying the weights placed on citations by, citations of, and topical similarity) aiming to return cases will be strong candidates for precedential reasoning. This will be accomplished via a new search-based algorithm that returns a ranked list of geometrically closest (using a search-based metric) cases as well as a map of the space, centered on a case of origin. Note that the cases are clickable and linked to the CourtListener Database. The "map" is generated by clicking on "View Case Network" and shows not only the relation of cases to the initial query, but also the relations of the closest cases to each other. This is a prototype for a larger effort.
